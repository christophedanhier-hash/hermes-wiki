## 📍 Détail par étape

### Étape 0 — 11/09 : Sombreffe → Colmar
- **Trajet** : 446 km (4-5h)
- **Départ** : 08:00 — arrivée vers 13:00
- Simple étape de route, balade en ville possible

### Étape 1 — 12→13/09 : Lac de Côme
- **Trajet** : Colmar → Côme = 353 km
- **Camping** : The Camp by Maggiore Suites ✅ (résa #32877)
- **Déplacements** : 🚲 vélos + ⛴️ bateaux
- **Incontournable** : Villa del Balbianello (Star Wars, Casino Royale), Varenna, Villa Carlotta
- **Billet bateau** : ~€15/pers.

### Étape 2 — 14→16/09 : Florence
- **Trajet** : Côme → Florence = 411 km
- **Camping** : Florence Sud ✅ (réf. #314185774)
- **⚠️ Réserver Uffizi et Accademia dès maintenant**
- **Déplacements** : 🚲 vélos + 🚌 tram
- **Incontournable** : Duomo, Galerie des Offices, David (Accademia), Piazzale Michelangelo

### Étape 3 — 17→18/09 : Pompéi
- **Trajet** : Florence → Pompéi = 484 km
- **Camping** : Fortuna Village Pompei ❌ (en demande)
- **Train** : Circumvesuviana (Herculanum 20 min, Naples 35 min)
- **Astuce** : Herculanum le matin (mieux conservé), Pompéi l'après-midi (train électrique)

### Étape 4 — 19→27/09 : Les Pouilles (base Monopoli)
- **Trajet** : Pompéi → Monopoli = 318 km
- **Camping** : Villaggio Turistico Monopoli ✅ (réf. #CD-0008)
- **Location voiture** : Sunny Cars 5 jours (21→25/09)
- **Jours forts** : Alberobello (trulli), Matera (Sassi), Lecce, Bari, Otrante, Gallipoli

### Étape 4b — 28/09 : Remontée vers Venise
- **Trajet** : Santa Irene = 377 km
- **Logement** : Area Sosta Camper ❌ (à confirmer)

### Étape 5 — 29/09→02/10 : Venise
- **Trajet** : Santa Irene → Venise = 546 km
- **Camping** : Europa Village, Cavallino-Treporti ✅
- **Vaporetto** : ~€7.50/trajet ou pass journalier ~€20
- **⚠️ Réserver Palais des Doges coupe-file**
- **Incontournable** : Place Saint-Marc, Palais des Doges, Pont du Rialto, Murano

### Étape 6 — 03→09/10 : Lac de Garde
- **Trajet** : Venise → Garde = 164 km (journée légère)
- **Camping** : San Francesco Camping Village ✅ (réf. #654795826)
- **Location voiture** : Sunny Cars 5 jours
- **Incontournable** : Sirmione, Grottes de Catulle, Riva del Garda, Malcesine

### Étape 7 — 10/10 : Lac de Garde → Colmar
- **Trajet** : 547 km
- **Logement** : Aire Schoenau ❌ (à confirmer)

### Étape 8 — 11/10 : Colmar → Sombreffe 🏠
- **Trajet** : 445 km — retour à la maison
